## 프로젝트 설명

워들(Wordle)이란, 2021년 Josh Wardle에 의해 만들어진 단어 게임입니다. [게임 사이트](https://www.nytimes.com/games/wordle/index.html) </br>
이 프로젝트는 워들이란 게임을 여러 방면 (알고리즘 연구, 비슷한 게임 개발) 으로 활용하고자 하는 사람들을 위해 제작된 파이썬3 라이브러리입니다.
